Para facilitar a observação dos resultados os programas foram salvos em diferentes pastas, uma para cada imagem.

Dessa forma cada pasta possui uma configuração de parâmetros (limiar, kernels, sigmas ...) otimizada para sua imagem.

O programa que executa o script principal é o "main.m".

Algumas versões mais simplificadas do pipeline foram testadas em imagens mais simples.

As imagens do banco de imagens disponibilizado pelo autor do artigo base estão em "imagens padrao".
